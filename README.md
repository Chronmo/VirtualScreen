# VirtualScreen（虚拟数据屏）

VirtualScreen，中文名“虚拟数据屏”，是一个公开开源的数据可视化大屏项目。

项目主要用于帮助大家从 0 到 1 学习如何自己动手制作一个完整的数据可视化大屏，包括页面布局、数据展示、图表设计、动态效果、工程化维护等内容。

## 项目定位

- 面向数据大屏初学者和前端学习者
- 以实战方式拆解数据可视化大屏开发流程
- 后续持续补充页面、组件、图表、交互和工程实践
- 保持代码清晰、结构易懂，方便学习和二次开发

## 当前状态

项目刚刚初始化，后续会逐步完善基础页面、图表组件、示例数据和开发文档。
